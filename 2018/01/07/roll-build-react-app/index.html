用Rollup打包React项目

什么是rollup?

​ 这是rollup官网给出的描述

Rollup 是一个 JavaScript 模块打包器，可以将小块代码编译成大块复杂的代码，例如 library 或应用程序

​ 简单的说， rollup就是一个js的打包器，它的优势在于可以通过Tree-shaking去掉多余的不需要的代码（webpack 2.x也实现了 Tree-shaking）
